## 📊 Project Overview
This project involved an end-to-end analysis of 7,000+ apps to identify growth drivers and user sentiment patterns. I developed a 4-page interactive Power BI dashboard to translate raw data into actionable business strategies.

## 💡 Key Insights
* **Market Leader:** Identified the **Game** category as the primary volume driver (~30B installs).
* **Technical Correlation:** Determined that user satisfaction peaks at a file size of **~80MB**, suggesting a preference for feature-rich applications.
* **Monetization Strategy:** Analysis showed that while **Free apps** drive 99% of engagement (reviews), **Paid apps** maintain slightly higher average quality ratings.
